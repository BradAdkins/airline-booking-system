# Airline Booking System

A Java-based airline booking system developed as a university software project.

## Features
- Create bookings
- Search bookings
- Cancel bookings
- Store booking data using CSV files
- REST API endpoints
- Docker support

## Technologies Used
- Java
- REST APIs
- Docker
- CSV file storage

## What I Learned
This project helped me improve my Java programming, backend development, API design, and Docker deployment skills.
